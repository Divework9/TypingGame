# v0.1 功能设计（添加键盘与配置文件）

## 版本目标
- 在 v0.0 基础上增加底部键盘可视化与按键闪烁反馈。
- 将关键配置模块化，降低主文件耦合。

## 功能范围
- 键盘渲染与按键闪烁逻辑。
- 键盘分层展示（letters / letters_numbers / full）。
- 配置文件拆分：
  - `conf/keyboard.py`
  - `conf/speed.py`
  - `conf/word_bank.py`

## 验收要点
- 游戏启动后具备键盘布局和按键状态数据。
- 速度参数来自 `conf/speed.py`。
- 英文/拼音词库来自 `conf/word_bank.py`。
- 各等级可加载对应键盘展示配置。
